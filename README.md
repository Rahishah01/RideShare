# RideShare

RideShare is a full-stack ridesharing web application built with Django.

The platform allows drivers to publish trips and passengers to search for
available rides, reserve seats, manage bookings, and view trip information.

## Features

- User registration and authentication
- User profile management
- Driver trip creation
- Ride search and filtering
- Seat booking system
- Duplicate-booking prevention
- Booking cancellation with seat restoration
- Passenger management
- Driver dashboard
- Custom administration portal
- User management
- Trip operations and mission control
- Booking management
- Dashboard statistics

## Technologies

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite for local development
- PostgreSQL for production
- WhiteNoise
- Gunicorn
- Render

## Project Status

The application is currently being prepared for public deployment.
